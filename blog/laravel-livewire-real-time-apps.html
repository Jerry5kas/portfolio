Building Real-Time Apps with Laravel Livewire: Complete Guide

AK

Azhagirishankar K

Full Stack Developer

Laravel Livewire allows you to build dynamic interfaces using only PHP, without writing JavaScript. This guide covers everything you need to build real-time applications.

1. Installation

composer require livewire/livewire

2. Create Livewire Component

php artisan make:livewire Counter

// app/Http/Livewire/Counter.php
class Counter extends Component
{
    public $count = 0;
    
    public function increment()
    {
        $this->count++;
    }
    
    public function render()
    {
        return view('livewire.counter');
    }
}

3. Real-Time Updates with Polling

<div wire:poll.1s>
    Current time: {{ now() }}
</div>

4. Events and Listeners

// Emit event
$this->emit('userUpdated', $userId);

// Listen to event
protected $listeners = ['userUpdated' => 'refreshUser'];

5. File Uploads

public $photo;

public function updatedPhoto()
{
    $this->validate([
        'photo' => 'image|max:1024',
    ]);
    
    $this->photo->store('photos');
}

6. Best Practices

  • Use wire:key for lists to maintain state
  • Optimize polling intervals
  • Use events for component communication
  • Validate data on the server side
  • Leverage Livewire's lifecycle hooks

Conclusion

Livewire enables rapid development of interactive applications while staying in the Laravel ecosystem. Perfect for building dashboards, admin panels, and real-time features.
