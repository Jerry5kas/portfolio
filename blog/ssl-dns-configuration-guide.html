SSL & DNS Configuration Guide: Secure Website Setup 2025

AK

Azhagirishankar K

Full Stack Developer

SSL certificates and proper DNS configuration are essential for secure, professional websites. This guide covers everything you need to set up SSL and configure DNS correctly.

1. DNS Configuration

Common DNS Records

  • A Record: Points domain to IP address
  • CNAME: Points domain to another domain
  • MX Record: Email server configuration
  • TXT Record: Verification and SPF records

Example DNS Setup

Type    Name    Value              TTL
A       @       192.0.2.1         3600
A       www     192.0.2.1         3600
CNAME   mail    mail.example.com  3600
MX      @       mail.example.com   3600

2. SSL Certificate Types

  • Domain Validated (DV): Basic validation
  • Organization Validated (OV): Business verification
  • Extended Validation (EV): Highest level
  • Wildcard: Covers subdomains

3. SSL Installation on Hostinger

  1. Log in to Hostinger control panel
  2. Navigate to SSL section
  3. Select "Let's Encrypt" or purchase SSL
  4. Install certificate
  5. Force HTTPS redirect

4. SSL Installation on GCP

# Using Google Cloud Console
1. Go to Cloud Load Balancing
2. Create SSL certificate
3. Attach to load balancer
4. Configure HTTPS redirect

5. Force HTTPS in Laravel

// In AppServiceProvider
public function boot()
{
    if (env('APP_ENV') === 'production') {
        URL::forceScheme('https');
    }
}

// In .htaccess (Apache)
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

6. DNS Propagation

  • DNS changes can take 24-48 hours
  • Use DNS checker tools to verify
  • Lower TTL before making changes
  • Clear DNS cache if needed

7. SSL Best Practices

  • Use strong encryption (TLS 1.2+)
  • Enable HSTS headers
  • Keep certificates updated
  • Monitor certificate expiration
  • Use certificate monitoring tools

Conclusion

Proper SSL and DNS configuration ensures your website is secure and accessible. Follow these guidelines to set up and maintain secure website infrastructure.
